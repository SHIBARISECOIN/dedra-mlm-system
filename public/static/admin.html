일일 ROI 정산 실행
회원이 구매한 투자 상품에서 매일 발생하는 ROI 수익을 정산하고, 그 수익을 기준으로 추천 체인 위로 유니레벨 보너스를 분배합니다.
⚙️ 정산 흐름 안내
1️⃣ 활성 투자 조회 → 각 투자의 D = 투자금 × 일일ROI% 계산
2️⃣ 투자 상품별 지급 패턴(주기, 시작일, 횟수 제한) 체크
3️⃣ 회원 지갑에 D(ROI 수익) 적립
4️⃣ D를 기준으로 Policy A(유니레벨 10단계) + Policy B(2세대 고정 직접 보너스) 분배
5️⃣ 직급 차이 보너스 포함 · 당일 정산 중복 실행 방지

엑셀에서 추출한 회원 JSON 데이터를 붙여넣고 Firestore에 일괄 등록합니다.
이미 존재하는 UID는 덮어쓰기(upsert)됩니다. 관리자 계정은 영향받지 않습니다.
임포트 후 회원 탭에서 확인하세요
🔄 구 시스템 ID 마이그레이션
현재 referralCode 필드에 잘못 들어간 로그인 ID를 username 필드로 이동시키고, referralCode는 UID 기반으로 새로 생성합니다.
⚠️ 이미 username 필드가 있는 회원은 건너뜁니다. 1회만 실행하세요.
referralCode → username 이동, 새 referralCode 생성
🔑 임포트 회원 비밀번호 일괄 초기화
username 필드가 있는 회원(임포트된 회원) 전체의 비밀번호를 아래 입력값으로 일괄 변경합니다.
⚠️ 기본값 000000으로 설정됩니다. 회원들이 첫 로그인 후 반드시 변경하도록 안내하세요.
